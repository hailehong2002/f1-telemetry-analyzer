# F1 Telemetry Racing Line Analyzer

A beginner-friendly autonomous systems project that analyzes Formula 1 telemetry data using Python and FastF1.

## Features

- Load real F1 session data
- Plot racing line
- Compare two drivers
- Visualize speed, throttle, and braking behavior
- Future: ghost replay and dashboard

## Tech Stack

- Python
- FastF1
- Pandas
- NumPy
- Matplotlib
- Plotly
- Streamlit

## Current Progress

- Basic telemetry loading
- Racing line visualization
- Driver comparison
